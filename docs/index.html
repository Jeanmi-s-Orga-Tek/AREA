Action REAction • Academic automation

AREA

Connect services, automate actions – an academic automation platform inspired by IFTTT and Zapier.

Automations orchestrate triggers, FastAPI exposes the public API, and web/mobile clients keep students in the loop with real-time workflows.

Voir la documentation Doxygen Explorer l'architecture

Plateforme AREA

  • Application server FastAPI
  • PostgreSQL + Docker orchestration
  • React/Vite web client
  • Mobile client (Android, WIP)

Architecture

Composants principaux

Application server

FastAPI expose les routes publiques (dont /about.json), gère les déclencheurs et orchestre les actions sur plusieurs services.

Web client

Interface React/Vite/TypeScript déployée via Docker, consommant /about.json pour afficher l'état des services et déclencheurs AREA.

Mobile client

Client mobile / Android (WIP) pour gérer les automatisations en mobilité et recevoir les notifications critiques.
